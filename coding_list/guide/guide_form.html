Form Guide

참고사항
  1. Design Selsect 태그 생성 : html상에 select 태그가 존재하면 ul,li,a(Design Selsect) 태그는 자동으로 동적 생성 합니다.
  2. Design Selsect 클릭시 값 전달받기 : Design Selsect에서 옵션 선택시 화면에 보이지않는 select 태그 option에 자동으로 selected가 삽입됩니다.
  1. Input text 또는 password의 경우 title 필수 입력 / id for로 연결 하지 않음
  1. checkbox 또는 Radio버튼은 input id와 label for로 필수 연결

01. Select Box

<div class="listTy1"> <select title="검색어 분류"> <option>전체</option> <option>제목</option> <option>내용</option> </select> </div>

02. Input Box

<input type="text" placeholder="검색어를 입력해 주세요" title="검색어 입력" /> <input type="text" placeholder="검색어를 입력해 주세요" title="검색어 입력" disabled="disabled" /> <input type="password" title="비밀번호 입력" />
